## 3. 감사업무

### 3-1. 트리 네비게이션
- [ ] FY → 클라이언트 → Phase → 계정과목 트리 구조 표시
- [ ] 노드 클릭 시 펼치기/접기 동작
- [ ] 계정과목 클릭 시 오른쪽 패널에 할일 목록 표시
- [ ] 클라이언트 클릭 시 오른쪽 패널에 요약 뷰 표시
- [ ] 트리 패널 접기/펼치기 (chevron 버튼)
- [ ] 접힌 상태에서 chevron 클릭으로 복원

### 3-2. 클라이언트 관리 (우클릭 컨텍스트 메뉴)
- [ ] FY 노드 우클릭 → "클라이언트 추가" 메뉴
- [ ] 클라이언트 노드 우클릭 → "이름 변경" 메뉴
- [ ] 클라이언트 노드 우클릭 → "Phase 추가" 메뉴
- [ ] 클라이언트 노드 우클릭 → "클라이언트 삭제" 메뉴 (확인 다이얼로그)
- [ ] Phase 노드 우클릭 → "계정과목 추가" 메뉴
- [ ] Phase 노드 우클릭 → "Phase 삭제" 메뉴

### 3-3. 계정과목 관리
- [ ] 계정과목 노드 우클릭 → "할일 추가" 메뉴
- [ ] 계정과목 노드 우클릭 → "계정과목 삭제" 메뉴

### 3-4. 할일 CRUD (리스트 뷰)
- [ ] 할일 목록 표시 (제목, 상태, 우선순위, 담당자, 마감일)
- [ ] "할일 추가" 버튼 클릭 시 할일 생성
- [ ] 상태 뱃지 클릭 시 상태 순환 변경 (미착수→진행중→검토대기→완료)
- [ ] 할일 우클릭 → 컨텍스트 메뉴 (제목 수정, 상태 변경, 삭제)
- [ ] 드래그 핸들로 할일 순서 변경 (리스트 뷰)

### 3-5. 할일 CRUD (칸반 뷰)
- [ ] 리스트/칸반 뷰 토글 버튼 동작
- [ ] 칸반: 4컬럼 (미착수/진행중/검토대기/완료) 표시
- [ ] 카드를 다른 컬럼으로 드래그 시 상태 변경
- [ ] 각 컬럼의 "+" 버튼으로 해당 상태의 할일 추가

### 3-6. 할일 상세 편집 (슬라이드 패널)
- [ ] 할일 카드 클릭 시 오른쪽 슬라이드 패널 오픈
- [ ] 제목 인라인 편집 (클릭하여 수정)
- [ ] 상태 버튼 그룹으로 변경
- [ ] 우선순위 버튼 그룹으로 변경
- [ ] 담당자 입력 필드
- [ ] 마감일 date picker
- [ ] 메모 textarea (자동 높이 조절)
- [ ] 참조 파일 경로 입력 필드
- [ ] 상태 변경 이력 타임라인 표시
- [ ] 소속 경로 (FY > Client > Phase > Account) 읽기 전용 표시
- [ ] "저장" 버튼 클릭 시 변경사항 저장 (dirty 상태에서만 활성)
- [ ] "삭제" 버튼 클릭 시 확인 후 삭제
- [ ] ESC 키로 패널 닫기
- [ ] 배경 오버레이 클릭으로 패널 닫기

### 3-7. 클라이언트 요약 뷰
- [ ] 트리에서 클라이언트 클릭 시 요약 뷰 표시
- [ ] 원형 프로그레스 (전체 진행률)
- [ ] 클라이언트명, 업종, 결산일, FY 표시
- [ ] 계정과목별 현황 테이블 (계정, Phase, 전체/완료, 진행률 바, 다음 마감일)
- [ ] 진행률 100% 행 연한 초록 배경
- [ ] 마감 초과 항목 빨간 텍스트 + "(초과)" 표시
- [ ] 테이블 행 클릭 시 해당 계정과목 할일 목록으로 전환
- [ ] 상태별 요약 도넛 차트 (SVG)
- [ ] 담당자별 업무 분배 (아바타 + 진행률 바)
- [ ] "엑셀 내보내기" 버튼 → CSV 다운로드
- [ ] "보고서 인쇄" 버튼 → window.print()

### 3-8. 탭 시스템
- [ ] 계정과목 선택 시: [할일 | 요청자료] 탭 표시
- [ ] 클라이언트 선택 시: [요약 | 요청자료] 탭 표시
- [ ] 탭 클릭으로 뷰 전환

### 3-9. 요청자료 (PBC) 탭
- [ ] 상단 요약: 전체 N건 중 수령완료 N건 + 진행률 바
- [ ] 테이블: 자료명, 계정과목, 요청일, 회신기한, 상태, 감사팀, 클라이언트, 비고
- [ ] 상태 뱃지 컬러 (미요청=회색, 요청완료=파랑, 수령완료=초록, 보완요청=주황)
- [ ] 수령완료 행 연한 초록 배경
- [ ] 회신기한 초과 항목 빨간 텍스트
- [ ] "추가" 버튼으로 요청자료 단건 추가
- [ ] "템플릿 가져오기" 버튼으로 일괄 추가
- [ ] 행 클릭 시 상세 편집 슬라이드 패널 오픈
- [ ] 상세 패널: 자료명, 계정과목 드롭다운, 상태 버튼, 요청일/회신기한, 담당자, 비고
- [ ] "엑셀 업로드" 버튼 클릭 시 엑셀 PBC 추적 모달 오픈

### 3-10. 엑셀 PBC 추적
- [ ] 파일 선택 다이얼로그 (xlsx/xls)
- [ ] 엑셀 원본 컬럼 테이블 렌더링
- [ ] 다중 시트 탭 전환
- [ ] 오른쪽 sticky 추적 컬럼: 수령여부(체크박스), 수령일(date picker), 완성도(○△×순환), 비고(팝오버)
- [ ] 수령 체크 시 오늘 날짜 자동 채움
- [ ] 완성도 클릭 순환: ○(초록) → △(주황) → ×(빨강) → 빈칸
- [ ] 상단 요약: 수령완료 N건/전체 N건 + 완성도별 카운트
- [ ] 같은 파일 재오픈 시 체크 상태 복원
- [ ] "내보내기" 버튼: 추적 컬럼 추가된 xlsx 다운로드
